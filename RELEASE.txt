COSMOS 5 Release Process

1. Make sure all tests are passing
2. Change versions to desired
3. Commit
4. Tag
5. Build Artifacts
   * Main Ruby Gem
   * Tool Ruby Gems
   * Demo Ruby Gem
   * Tool-Common Npm Package
   * 8 Docker Containers
6. Publish Artifacts
   * Ruby gems to rubygems.org
   * Tool-Common to Npm
   * Docker Containers to Docker Hub (Or have docker hub build on tag? Probably not, but how to build for ARM?)
7. Change versions to next beta
8. Commit
9. Write-up Release Notes on Github
10. Release Notes on Website
